# 🌱 AgriAI Crop Prediction

AgriAI Crop Prediction is a smart agriculture web application designed to assist farmers and agricultural planners in making data-driven decisions. The platform uses AI-based insights to provide crop recommendations, soil health analysis, weather data visualization, and yield predictions.

---

## 🚀 Features

- 🌾 Crop prediction based on environmental conditions  
- 🌱 Soil health monitoring and analysis  
- 🌦️ Weather insights dashboard  
- 📊 Yield prediction with interactive charts  
- 💡 Smart farming recommendations  
- 🎨 Clean and responsive user interface  

---

## 🛠️ Tech Stack

- React.js, TypeScript  
- Vite  
- Tailwind CSS  

## ⚙️ Setup

bash
git clone https://github.com/your-username/AgriAI-Crop-Prediction.git
cd AgriAI-Crop-Prediction
npm install
npm run dev
